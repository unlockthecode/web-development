Learning Hypertext Markup Language (HTML)
──────────────────────────────

Reference Tutorial:
- https://youtu.be/mJgBOIoGihA - HTML Full Course for Beginners by Dave Gray 

──────────────────────────────
General Notes
──────────────────────────────
- index.html is usually served by web servers at the root, but not strictly required.
- Use the W3C Markup Validation Service (validator.w3.org) to check your HTML.
- Always start with <!DOCTYPE html> and add lang attribute to <html>.
- <meta charset="UTF-8"> should be inside <head>.

──────────────────────────────

- Live Server (for instant browser preview)
- HTML CSS Support (for CSS help)

──────────────────────────────
HTML Structure Overview
──────────────────────────────
<html lang="en">
    <head>
        <!-- Metadata, title, links, scripts, styles -->
    </head>
    <body>
        <!-- Visible content -->
    </body>
</html>

──────────────────────────────
Key Elements & Their Purposes
──────────────────────────────
<head>      : Metadata, links, scripts, styles (not visible)
<header>    : Top section of page/section (logo, nav, etc.)
<footer>    : Bottom section (contact, copyright, nav)
<nav>       : Navigation links (use aria-label for accessibility)
<main>      : Main content of the page
<aside>     : Sidebar or related content
<section>   : Thematic grouping of content
<article>   : Independent, self-contained content
<div>       : Generic block-level container
<span>      : Generic inline container

──────────────────────────────
Text & Semantics
──────────────────────────────
- Headings: <h1> (only one per page), <h2>, <h3>, ...
- Paragraphs: <p> (use one per logical block of text)
- Horizontal rule: <hr> (topic change)
- Line break: <br> (use sparingly)
- Block-level elements: <div>, <p>, <h1>-<h6>, <ul>, <ol>, <li>, <section>, <article>, <header>, <footer>, <form>, <table>, <blockquote>
- Inline elements: <span>, <a>, <strong>, <em>, <img>, <input>, <button>, <code>, <small>, <sub>, <sup>
- Semantic tags: <em>, <strong>, <abbr>, <address>, <mark>, <time>, <code>, <!-- Comment -->

──────────────────────────────
Lists
──────────────────────────────

<ul>      : Unordered list

Links
──────────────────────────────
<a href="url">Text</a>           : Basic link
<a href="#id">Jump to section</a>: Internal link
<a href="url" target="_blank">   : Open in new tab

──────────────────────────────
Forms & Inputs
──────────────────────────────
# https://httpbin.org/post can be used to test form submissions.
<form action="https://httpbin.org/post" method="post">
    <fieldset>
        <legend>Group Name</legend>
        <label for="name">Name:</label>
        <input type="text" id="name" name="name" required>
        <!-- More inputs -->
    </fieldset>
    <input type="submit" value="Submit">
</form>

Input types: text, email, tel, number, date, password, checkbox, radio, submit, reset, file, hidden, button, color, range, search, time, url, week, month
Important attributes: type, id, name, value, placeholder, required, pattern, minlength, maxlength, min, max, step, autocomplete

<select id="color" name="color" multiple>
    <option value="red">Red</option>
    <option value="green">Green</option>
</select>

<textarea name="message"></textarea>

──────────────────────────────
Styling & CSS Tips
──────────────────────────────
- Use classes for indentation:
    <p class="indented">Indented text</p>
    .indented { margin-left: 20px; }
- Remove default <p> spacing inside a container:
    .container p { margin: 0; padding: 0; line-height: 1.4; }
- Add space after last <p> in a section:
    .section p:last-child { margin-bottom: 1em; }
- Keep link color consistent:
    a, a:visited, a:active, a:focus { color: whitesmoke; }

──────────────────────────────
Common Problems & Solutions
──────────────────────────────
- Use <span title="..."> for tooltips instead of <abbr> if needed.
- Use class for indentation instead of &nbsp;.
- Use CSS to control spacing between <p> and sections.
- Use target="_blank" for links to open in new tab.
- Change stylesheet link in <head> for different pages.
